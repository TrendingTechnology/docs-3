# Introduction

Bothell.js is a Foundational React UI Component Library. It is open-source, lightweight and responsive. Bothell.js only allows minimal customization to the original styles, such as colors and fonts. The goal of this library is the build simple and elegant applications with consistent theming and design across all components of the application, increase productivity and decrease overall time spent on design tweaks.
